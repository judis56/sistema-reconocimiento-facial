# Sistema Inteligente de Asistencia con Reconocimiento Facial

## Descripción
Sistema desarrollado para automatizar el control de asistencia mediante reconocimiento facial utilizando Python e inteligencia artificial.

## Funciones
- Detección y reconocimiento facial
- Registro automático de asistencia
- Almacenamiento de datos en archivos CSV
- Interfaz gráfica para administración

## Tecnologías utilizadas
- Python
- OpenCV
- CSV
- Inteligencia Artificial

## Archivos principales
- `interfaz.py`
- `dashboard.py`
- `asistencia.csv`
